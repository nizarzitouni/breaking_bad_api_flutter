# Breaking_bad_api_app

A new Flutter project that uses the breaking bad api to display Infos

This App was built using :

-[flutter_offline : ^4.2.2 ](https://pub.dev/packages/flutter_offline) Package.

-[animated_text_kit: ^1.0.1](https://pub.dev/packages/animated_text_kit) Package.

-[flutter_bloc: ^8.0.1](https://pub.dev/packages/flutter_bloc) Package.

-[dio: ^4.0.6](https://pub.dev/packages/dio) Package.
